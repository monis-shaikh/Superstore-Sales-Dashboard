🚀 Superstore Sales Dashboard | Power BI Project

📌 Overview
This project presents a comprehensive Business Intelligence solution built using Power BI to analyze and visualize sales data from a retail superstore.
The dashboard transforms raw transactional data into actionable insights by focusing on:
•	Sales performance analysis
•	Product-level insights
•	Shipping and logistics optimization
It is designed to support data-driven decision-making through interactive and visually intuitive dashboards.
________________________________________
🎯 Problem Statement
To design and develop interactive dashboards that provide deep insights into:
•	Sales trends and profitability
•	Product performance and discount patterns
•	Shipping modes and operational efficiency
________________________________________
📊 Key Features
•	📈 Interactive and dynamic dashboards
•	🔍 Drill-down analysis across multiple dimensions
•	📅 Time-based filtering (Year, Quarter, Month)
•	🌍 Region-wise and market-wise insights
•	📦 Category and sub-category level analysis
•	🚚 Shipping mode and cost analysis
________________________________________

📂 Dataset Information
•	Dataset Name: Superstore Sales Dataset
•	Source: Kaggle
•	Time Period: 2011 – 2014
•	Records: 50,000+ transactions
🔗 https://www.kaggle.com/datasets/laibaanwer/superstore-sales-dataset
________________________________________
🧠 Dashboard Insights
📈 Sales Analysis
•	Total Revenue: 13M+
•	Total Profit: 1.47M+
•	Peak sales observed in March–April
•	Technology category contributes highest profit
________________________________________
📦 Product Analysis
•	Office Supplies dominate in order quantity
•	High discounts observed in Furniture category
•	Sub-category level performance insights
________________________________________
🚚 Shipping Analysis
•	Standard Class is the most preferred shipping mode
•	Same Day delivery has minimal usage
•	APAC region incurs highest shipping cost
